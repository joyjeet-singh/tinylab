# requirements-colab.txt
#
# The Colab (GPU) environment. Companion to requirements.txt, which is the
# Mac/CPU pin (torch 2.2.2, capped by the 2017 MacBook + Python 3.11).
#
# Why the two files differ, and why that's fine:
#   The Mac pin is a hardware limitation, not a scientific choice. Colab ships
#   its own CUDA-matched torch build, and forcing a different version is the
#   fastest way to break a session. So torch is deliberately NOT pinned here.
#   Record which requirements file a run used in its manifest, so any result
#   can be traced back to the environment that produced it.
#
# Usage in Colab:
#   !pip install -q -r requirements-colab.txt
#
# NOTE: torch / torchvision are intentionally absent. Use Colab's preinstalled
# GPU build. Verify it before running anything:
#   import torch; print(torch.__version__, torch.cuda.is_available())
